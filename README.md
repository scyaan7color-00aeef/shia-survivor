# シア・サバイバー

押し寄せる敵をかわして生き延びる、ブラウザで遊べる見下ろし型サバイバーアクション。

▶ **遊ぶ:** GitHub Pages（公開後にURLをここへ）

## 遊び方
- 移動：WASD ／ 矢印キー（スマホは画面をドラッグ）
- 攻撃は自動。移動と回避に集中しよう
- ジェムを拾うとレベルアップ、4択で強化
- 一時停止中はコインでショップを利用

## 特徴
- 攻撃アーキタイプ6職（弾／周回球／斬撃／ポーション爆弾＋毒沼／チョーク ほか）をキャラ選択で切替
- 武器進化・転生（星屑）による周回強化
- ボスラッシュ、6ゾーンの背景切替
- ローカルランキング・プレイ履歴・モンスター図鑑

## 技術
- Pure HTML5 Canvas + Vanilla JavaScript（ビルド不要・依存ゼロ）
- ローカルストレージにセーブ
